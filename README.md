# glmu 桂林医科大学课程表提取
一、现开发了针对桂林医科大学教务系统网站的，课程表json文件提取插件

二、该插件适配了  拾光课程表  的教务系统导入功能，使用更方便！！

注：本项目大部分由Deepseek编写，我仅当了工具人

一、桂医课程表提取插件 (桂医课程表提取插件 文件夹)

这是一款专门针对桂林医科大学教务系统网站的课程表json文件提取的，浏览器插件。目前使用方法较为复杂。

使用方法：1.下载整个文件夹的zip压缩包  并在本地解压

2.打开浏览器扩展管理界面 (以edge为例) 再打开 开发人员模式

3.点击 加载解压缩的扩展  选择刚刚下载并解压好的文件夹  再点击 选择此文件夹

4.扩展管理界面，打开扩展并重新加载(保险操作)

5.打开桂医教务系统并登陆  进入主页就行

6.点击  右上角拼图图标  再点击该插件

7.按照提示 选好相关信息后 点击提取

8.提取的课程表json文件 自动下载在 浏览器下载路径内

9.自行导入  支持json文件导入的 课程表软件

二、该插件适配了  拾光课程表  教务系统导入功能！！！

该插件适配代码见  GLMU 文件夹

下载 拾光课程表 后更新教务系统列表  就可直接使用
